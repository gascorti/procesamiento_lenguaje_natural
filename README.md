# **PROCESAMIENTO DEL LENGUAJE NATURAL**

**Modelos Clásicos, Representaciones Vectoriales y Modelos Basados en Deep Learning** 

Este repositorio reúne implementaciones, experimentos y notebooks educativos orientados a introducir y profundizar en técnicas de Procesamiento de Lenguaje Natural (PLN). Incluye desde métodos clásicos basados en bolsa de palabras, TF–IDF, Naïve Bayes, similaridad semántica, hasta modelos más avanzados como RNN, LSTM, GRU, y estrategias de generación de texto.

![PNL](imagenes/PNL.png)

**Contenido del repositorio**
1. Representaciones vectoriales clásicas.
2. Modelos de clasificación como Naïve Bayes Multinomial y ComplementNB.
3. Embeddings y Gensim.
4. Modelos de lenguaje (RNN / LSTM)
